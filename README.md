# AI Quality Engineering (QE) Coursework

This repository contains my comprehensive coursework and projects from the **AI QE Course at Alex Academy**. It covers the full spectrum of AI testing, from basic data structures to advanced RAG (Retrieval-Augmented Generation) frameworks.

## 📂 Project Structure
- **AI Class #01-24:** Weekly assignments covering NLP, Attention Mechanisms, and Model Evaluation.
- **AI QE Framework:** A production-ready RAG pipeline including:
  - Data validation and semantic filtering.
  - Performance testing and reporting.
  - UI-based interaction and chat logging.
- **Hardware/Voice Integration:** Experiments with OpenAI whisper and multi-tool voice agents.

## 🛠 Tech Stack
- **Languages:** Python
- **Tools:** Git, Jira (JQL), Postman
- **AI Concepts:** RAG, Fine-tuning, RLHF, Attention mechanisms, Vector databases (FAISS).

## 🚀 Key Frameworks
The **AI QE Framework** folder contains my primary project, featuring automated validation scripts and semantic analysis tools for testing LLM outputs.
