--------------------------------------------------------------------------------
1. PROJECT OVERVIEW
--------------------------------------------------------------------------------
The goal of this assignment was to build a multilingual "Grammar Critic" capable 
of identifying missing sentence components. The system uses deep learning 
models to parse the syntax of a sentence and check for the presence of 
subjects, verbs, and predicates.

--------------------------------------------------------------------------------
2. FILE BREAKDOWN
--------------------------------------------------------------------------------
- LOGIC SCRIPTS (The Engines):
  * stanza_en.py / stanza_ru.py: Uses the Stanza library to download and run 
    neural pipelines for English and Russian. 
  * UDPipe_ML.py: A unified script using pre-trained .udpipe models for 
    Russian, English, and Ukrainian. It includes a custom regex-based 
    language detector.

- DATA INPUTS (Raw Text):
  * sentences_en.txt / sentences_ru.txt: Language-specific test sentences.
  * sentences.txt: A mixed-language file containing English, Russian, and 
    Ukrainian text.

- AUDIT REPORTS (CSV Output):
  * errors_en.csv / errors_ru.csv: Reports generated by the Stanza scripts.
  * errors.csv: The final consolidated report identifying sentence status (OK/BAD) 
    and specific error codes.

--------------------------------------------------------------------------------
3. KEY LOGIC: NLP AUDITING
--------------------------------------------------------------------------------
- DEPENDENCY PARSING: 
  The scripts look for specific tags like "nsubj" (subject) and "root" (predicate) 
  to determine if a sentence is grammatically complete.

- MATHEMATICAL FILTERING: 
  A custom function identifies if a string is actually a math equation 
  (e.g., "2 * 2 = 4") based on symbol density, labeling it as a MATH::SENTENCE 
  error rather than a grammar error.

- ERROR CODES:
  * NO::SUBJECT: Missing a nominal subject.
  * NO::VERB / NO::PREDICATE: Missing the core action or root of the sentence.

--------------------------------------------------------------------------------
4. EXECUTION
--------------------------------------------------------------------------------
To run the multilingual analyzer:
    python UDPipe_ML.py

To install the NLP requirements:
    pip install stanza ufal.udpipe
